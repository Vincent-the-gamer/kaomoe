# Usage
## Use `CLI` to generate Kaomoji:

```shell
pnpm run dev
```

## Use `web server` to generate Kaomoji:

```shell
pnpm run server
```

## Refresh Kaomiji List
```shell
pnpm run fetch
```

## Build
```shell
pnpm run build
```
